# Fake News Detection System

## Description
A machine learning-based project that classifies news as real or fake.

## Technologies Used
- Python (Core Programming Language for ML)
- TF-IDF Vectorizer (Text Feature Extraction)
- Passive Aggressive Classifier (Machine Learning Algorithm)
- Streamlit (Web Application Framework)

## Features
- User input news text.
- Predicts whether news is fake or real.
